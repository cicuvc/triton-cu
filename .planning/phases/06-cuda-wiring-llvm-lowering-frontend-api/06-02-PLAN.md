<objective>
Wire shared-memory arguments through the CUDA compilation pipeline: `infer_result()` builds degenerate `SharedTensorParameter` for template deduction, `_pre_compile_extern_calls()` consumes Phase-5 spec JSON to build real `SharedTensorParameter` at the llir stage, the `ttg.extern_call_arg_spaces` module attribute carries per-operand memory-space info to the C++ lowering, and `BuildSharedTensor` applies the clang addrspace qualifier for correct `ptr addrspace(3)` emission.

Purpose: The suspended-compiler single-parse path must handle shared args without triggering a second parse, and the compiled bitcode callee signature must natively accept `ptr addrspace(3)` to avoid addrspacecast at call sites.
Output: Modified `compiler.py` (infer_result shared branch, _pre_compile_extern_calls shared branches in both spec loops, arg_spaces metadata + module attr emission) and `clang_compiler.cc` (BuildSharedTensor addrspace wrapping per D-15).
</objective>
